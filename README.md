# Result Management System

## Overview
A Python-based Result Management System developed to manage student academic records efficiently.

## Features
- Add Student Records
- Update Results
- Delete Records
- Search Student Data
- Calculate Grades
- Generate Result Reports

## Technologies Used
- Python
- File Handling
- Object-Oriented Programming (OOP)
